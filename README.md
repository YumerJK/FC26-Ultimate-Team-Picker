# FC 26 Ultimate Picker

FC 26 için kararsız kaldığında hızlıca takım seçmeni sağlayan tek sayfalık mini uygulama.

## Kullanım

1. `index.html` dosyasını tarayıcıda aç.
2. Lig filtresini seç.
3. **TAKIMI BELİRLE** butonuna bas.
4. Rastgele seçilen takımın hücum/savunma değerlerini ve yıldız oyuncusunu gör.
